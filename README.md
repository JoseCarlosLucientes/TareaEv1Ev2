
# Habana Hotel

Este proyecto es una aplicación de gestión de reservas para un hotel. Permite a los usuarios realizar nuevas reservas y ver las reservas existentes.

## Características

- **Nueva Reserva**: Permite a los usuarios agregar nuevas reservas con información detallada, como nombre, teléfono, tipo de evento, número de jornadas, etc.
- **Ver Reservas**: Permite a los usuarios ver todas las reservas existentes en una tabla con funcionalidad de búsqueda y filtrado.
- **Estilo Personalizable**: La aplicación proporciona opciones para cambiar el estilo de la tabla de reservas y la cabecera.

## Tecnologías Utilizadas

- Java
- Swing (Biblioteca GUI para Java)
- JUnit (Framework de pruebas unitarias para Java)

## Estructura del Proyecto

El proyecto consta de las siguientes clases principales:

1. **Principal**: La clase principal que inicia la aplicación y proporciona la interfaz de usuario principal.
2. **Reservas**: Clase para gestionar la creación y visualización de reservas.
3. **Listado**: Clase para mostrar una lista de reservas en una tabla.
4. **PrincipalTest**: Clase de pruebas unitarias para la clase `Principal`.
5. **ReservasTest**: Clase de pruebas unitarias para la clase `Reservas`.
6. **ListadoTest**: Clase de pruebas unitarias para la clase `Listado`.

## Cómo Ejecutar

1. Clona este repositorio en tu máquina local.
2. Abre el proyecto en tu IDE de Java preferido.
3. Ejecuta la clase `Principal` para iniciar la aplicación.
